## What is D&D 5e Character Sheet?

D&D 5e Character Sheet is a standalone HTML character sheet intended to be opened in a web browser. It is aimed at tabletop players who want a simple way to track a character without depending on a server, installer, or online account.

The app is built to streamline D&D 5e character management both during play and while preparing a build. It pairs automatic stat handling with organized sections for the pieces that often expand over time, such as inventory, spells, companions, notes, and other tracking areas.

## Features

- Single-file HTML app that works fully offline
- D&D 5e character sheet with automatic calculations
- Save and restore directly in the browser
- Portrait upload and image handling
- Character creation wizard for race, class, background, and equipment
- Multiclass support with spell slots, features, and resources
- Dedicated pages for inventory, spells, companions, notes, and more
- Print-friendly layout with PDF-oriented output support

## Installation

1. Download or clone the repository.
2. Open the HTML file directly in a modern web browser.
3. No server setup is required for normal use.

If you want to keep a local copy, place the HTML file in any folder and open it from disk whenever you need it.

---

## Usage

Open the file in your browser first, then use the character creation wizard to start a new sheet or load a saved one. As you fill in race, class, background, equipment, and other details, the sheet updates the core values automatically.

Typical workflow:

1. Create a new character in the wizard.
2. Enter or adjust class, level, and multiclass details.
3. Add inventory, spells, companions, and notes on the dedicated pages.
4. Save your work through the browser-based restore flow.
5. Use the print-friendly layout when you want a paper or PDF copy.

---

## Configuration

Most options live inside the sheet itself and are stored through the browser workflow. Because this is a single-file app, there is no separate config directory to maintain.

If you need to keep changes, store the HTML file and your saved browser data in the same browser profile you normally use for the sheet.

## FAQ

**Does it need an internet connection?**  
No. The app is designed to run offline in a browser.

**Do I need to install anything?**  
No installation is necessary. Open the HTML file and begin using it.

**Where are saves kept?**  
Saves are managed through the browser-based save and restore flow.

**Can I print my character?**  
Yes. The layout is built to be print-friendly and works well for PDF output.

**What if my browser does not save data correctly?**  
Try another browser or browser profile, then open the file again and repeat the save process.
